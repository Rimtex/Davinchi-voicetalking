
Голос Microsoft Pavel Mobile предназначен для использования в приложениях из магазина Windows Store.
По умолчанию к голосу нельзя обратиться при помощи функций SAPI 5.
Однако, есть простой способ решить данную проблему.
("pavel_x86.reg" для 32-битной версии и "pavel_x64.reg" для64-битной версии),
кликните правой кнопкой мыши на имени файла и выберите пункт "Слияние" в контекстном меню. 
Данные о голосе добавятся в системный реестр Windows, и голос можно будет использовать в программе.
